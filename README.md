# sportgym - Página web responsive (demo)

Pequeña página estática creada en HTML/CSS/JS (mobile-first) para un gimnasio.

Cómo ver:
1. Abrir `index.html` en un navegador moderno.
2. (Opcional) Instalar la extensión "Live Server" en VSCode y ejecutar para ver recargas automáticas.

Qué incluye:
- `index.html` - estructura en español
- `css/styles.css` - estilos responsivos usando Grid y Flexbox
- `js/script.js` - menú móvil y scroll suave
